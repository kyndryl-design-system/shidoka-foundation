## Usage

### SCSS Mixins

```css
@use '@kyndryl-design-system/shidoka-foundation/scss/mixins/grid.scss';

.your-css-selector {
  @include grid.grid(
    $align-left: false,
    $no-max: false,
    $no-gap: false,
    $full-bleed: false,
    $compact: false
  );
}

.your-child-css-selector {
  @include grid.grid-col(num_cols);
}
```

### Utility Classes

#### Import to SCSS

```css
@use '@kyndryl-design-system/shidoka-foundation/scss/utility/grid.scss';
```

#### Import to CSS

```css
@import '@kyndryl-design-system/shidoka-foundation/css/grid.css';
```

#### Import to JS

```js
import '@kyndryl-design-system/shidoka-foundation/css/grid.css';
```

#### Example HTML

```html
<div class="kd-grid">
  <div class="kd-grid__col--sm-4 kd-grid__col--md-8 kd-grid__col--lg-12">
    12 cols @ max, xl, lg, 8 cols @ md, and 4 cols @ sm
  </div>
</div>
```

#### Column class name schema

```
kd-grid__col--${breakpoint}-${num_cols}
```

## Breakpoints

| Breakpoint   | Value (rem/px) | Columns | Gap  |
| ------------ | -------------- | ------- | ---- |
| Small (sm)   | 20 / 320       | 4       | 16px |
| Medium (md)  | 42 / 672       | 8       | 32px |
| Large (lg)   | 74 / 1184      | 12      | 32px |
| X-Large (xl) | 82 / 1312      | 12      | 32px |
| Max (max)    | 99 / 1584      | 12      | 32px |

## Additional Utility Classes

### Container

| Description                     | Class               |
| ------------------------------- | ------------------- |
| Left align grid container       | kd-grid--align-left |
| Remove grid container max-width | kd-grid--no-max     |
| Remove column gap               | kd-grid--no-gap     |
| Compact column gap              | kd-grid--compact    |
| Full bleed/override page gutter | kd-grid--full-bleed |

## Page Gutter

A page gutter should be applied once globally in the application on the `<body>` tag or similar, or by using the shidoka-applications [UI Shell component](https://shidoka-applications.netlify.app/?path=/docs/global-components-ui-shell--docs). There is also an option to negate the page gutter if you need to make an element "full bleed".

| Utility Class                    | CSS Variable              | Value                                      |
| -------------------------------- | ------------------------- | ------------------------------------------ |
| kd-spacing--page-gutter          | --kd-page-gutter          | padding-left/right: 16px @ sm, 32px @ md+  |
| kd-spacing--page-gutter-negative | --kd-page-gutter-negative | margin-left/right: -16px @ sm, -32px @ md+ |

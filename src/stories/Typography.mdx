## Usage

### SCSS Mixins

```css
@use '~@kyndryl-design-system/shidoka-foundation/scss/mixins/typography.scss';

.your-css-selector {
  @include typography.type-hero-01;
}
```

### Utility Classes

#### Import to SCSS

```css
@use '~@kyndryl-design-system/shidoka-foundation/scss/utility/typography.scss';
```

#### Import to CSS

```css
@import '@kyndryl-design-system/shidoka-foundation/css/typography.css';
```

#### Import to JS

```js
import '@kyndryl-design-system/shidoka-foundation/css/typography.css';
```

#### Example HTML

```html
<h1 className="kd-type--hero-01 kd-type--weight-light">Hero-01 Light</h1>
```

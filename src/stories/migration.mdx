# Migration Guide

Breaking Changes

## 2.0

- Icon component removed
  - Use [Shidoka Icons](https://github.com/kyndryl-design-system/shidoka-icons) by importing the custom SVG files directly from this library.
- Colors
  - Removed the old color tokens which have been completely reworked and replaced with a new set of tokens.
- All components moved to shidoka-applications.
  - Tag prefixes changed from `kd-` to `kyn-`.
